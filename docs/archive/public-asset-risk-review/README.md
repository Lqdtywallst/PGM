# Public Asset Risk Review Archive

Unverified binary files were removed from `site/` on 2026-05-17 because their
public-use origin or suitability was not clear enough for the current
production site.

They should not be restored or referenced by public HTML, CSS, JavaScript,
metadata, sitemap entries, or production renderers unless ownership or license
has been reviewed and approved.

Current public replacements use original generated assets created for Dynasty
Prestige, with no third-party logos, no recognizable vehicle badges, no
license plates, and no identifiable people.

`site/images/fleet/` was intentionally left untouched because the project
confirmed those assets are approved for use.
